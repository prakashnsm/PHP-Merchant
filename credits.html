PHP Merchant is developed and maintained by Instinct Entertainment. Some of the code was borrowed from and designed for our WordPress e-Commerce Plugin project.

We're building a community around this project so that other developers can contribute code and ideas. See the page about Contributing for more information.

Kudos to the awesome active_merchant project for giving us inspiration to create this payment gateway library for the PHP community.

The thoroughness and outline of documentation is influenced by CodeIgniter's User Guide. Of all PHP projects out there, CodeIgniter probably has one of the best documentations.

Other tools that we use to manage the project and documentation are:

  • github: awesome git hosting service.
  • jekyll: manage and generate documentation using templates.
