PHP Merchant has some utility functions that are used throughout its code base.

  • phpme_map( $from_array, $map, $return_type = 'Array' )

    What this function does is mapping new array key to the same values in $from_array.

    Arguments:

    • $from_array: The array that you want to extract the values from.
    • $map: An associative array containing pairs of $new_key => $old_key.
    • $return_type: Defaults to 'Array'. You can make this function returns an object by specifying $return_type as 'Object'.

    Return: A new array with the new keys containing values from the $from_array.
