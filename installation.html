Drop-in installation

Installing PHP Merchant is easy. Simply download the source and put it in a folder within your PHP site. You can download PHP Merchant from Github.

The downloaded package will have the following sub-folders:

  • common: contains common classes and functions that are used in all payment gateway classes. You only need to require 'common/php-merchant.php' to include all these necessary components.
  • gateways: contains individual gateway implementations. When you need to use a payment gateway, simply require 'gateways/{gateway-name}.php'.
  • tests: contains unit tests. This might be useful if you want to check the stability of current code base (it should pass all tests).

Cloning our github repository

Git is the version control system that we use to manage our codebase. If you know your way around git, you can easily get updates via github with only a few commands.

To clone our read-only repository:

{% highlight bash linenos %} cd /path/to/your/php/site git clone git://github.com/instinct/PHP-Merchant.git{% endhighlight %}

You should only use code on our master branch for production sites. Code on master is guaranteed to have better unit-test coverage and integration testing.

Run this every time you want to get update from our master branch:

{% highlight bash linenos %} cd /path/to/your/php/site/PHP-Merchant git pull origin master{% endhighlight %}
